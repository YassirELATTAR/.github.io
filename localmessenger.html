I am currently working on a personal project in my spare time, which involves creating a desktop application to facilitate communication between school staff without the need for an internet connection. The goal of this project is to provide a more efficient and reliable way for staff members to communicate important information, such as schedule changes or announcements, even in areas with poor or no internet connectivity. I am excited to be using my programming skills to make a tangible impact in the education sector.
PS: This design is primary using HTML/CSS just to test basic operations.
